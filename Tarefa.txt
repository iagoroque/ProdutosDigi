Tarefa: Desenvolvimento de Página Web para Cadastro de Produtos com Funcionalidade CRUD e Exibição em Grade
Objetivo:
Desenvolver uma página web que permita realizar o cadastro de produtos utilizando operações de CRUD (Create, Read, Update, Delete), com uma grade para exibição e gerenciamento desses produtos em tempo real.

Requisitos:

Cadastro de Produtos:

Criar uma interface que permita ao usuário:
Inserir novos produtos com campos como nome, descrição, preço e quantidade.
Editar as informações de produtos já cadastrados.
Excluir produtos do cadastro.
Visualizar todos os produtos cadastrados em uma grade ou tabela.
Funcionalidade CRUD:

Implementar as seguintes operações de CRUD:
Create (Criar): Adicionar novos produtos ao sistema.
Read (Ler): Exibir todos os produtos cadastrados em uma grade/tabela.
Update (Atualizar): Editar as informações dos produtos existentes.
Delete (Excluir): Remover produtos do sistema.
Exibição em Grade:

A página deve incluir uma grade ou tabela que exiba todos os produtos cadastrados. A grade deve:
Mostrar todos os produtos com detalhes como nome, descrição, preço e quantidade.
Ser atualizada automaticamente após cada operação de CRUD.
Oferecer funcionalidades opcionais como ordenação e paginação.
Validação e Persistência de Dados:

Implementar validação de dados para garantir que apenas informações válidas sejam inseridas.
Os dados dos produtos devem ser armazenados de forma persistente, utilizando um banco de dados ou uma estrutura de dados equivalente.
Interface do Usuário:

Desenvolver uma interface amigável e responsiva, garantindo uma boa experiência de usuário.
Incluir mensagens de feedback para operações bem-sucedidas ou erros durante o cadastro, edição ou exclusão de produtos.
Entrega:

Código-fonte completo da aplicação web.
Instruções detalhadas para rodar o projeto localmente, incluindo a configuração do banco de dados ou outras dependências.
Documentação explicando a implementação das funcionalidades CRUD e como a grade de visualização foi configurada.
Critérios de Avaliação:

Funcionalidade completa e correta das operações de CRUD.
Qualidade do código e organização do projeto.
Usabilidade e design da interface.
Eficiência na atualização e exibição dos dados na grade.
Implementação de validação de dados e tratamento de erros.
